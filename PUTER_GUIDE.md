# How to Use Puter (Free AI)

## Two Ways to Select Provider

### Method 1: Settings (Default for ALL conversations)
1. Go to **Settings** → **API Keys** tab
2. Click **Add Key**
3. Select **☁️ Puter (Free Claude)**
4. Enter name: "Free Puter"
5. Click **Add Key** (no API key needed!)
6. Go to **Preferences** tab
7. Set **Default Provider** to **Puter**

Now ALL new conversations use Puter by default.

### Method 2: Chat Interface (Per conversation)
1. In any chat, look at the top bar
2. Click the provider dropdown (shows current provider)
3. Select **☁️ Puter**
4. Select a model from the model dropdown

This only affects the current conversation.

## Features

✅ **All File Types** - Attach any file (images, PDFs, code, etc.)
✅ **Streaming** - See responses in real-time
✅ **Thinking Indicator** - Shows when AI is processing
✅ **No Setup** - Works immediately

## Difference Between Settings & Chat

**Settings Provider:**
- Sets the default for NEW conversations
- Applies globally
- Requires adding provider first

**Chat Provider:**
- Changes current conversation only
- Instant switch
- Overrides default

Both work together - Settings sets the default, Chat lets you override per conversation.
