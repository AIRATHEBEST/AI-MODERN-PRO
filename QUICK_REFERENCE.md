## 🚀 Quick Start Examples

### Example 1: Multi-Agent Research
```
1. Press ⌘K
2. Type "Research Agent"
3. Or go to Agents tab
4. Enter: "Research AI trends in 2025 and create a report"
5. Watch agents collaborate
```

### Example 2: Background Task
```
1. Go to Tasks tab
2. Schedule a task
3. Set priority (urgent/high/medium/low)
4. Monitor progress
5. View results when complete
```

### Example 3: Code Generation
```
1. Press ⌘K
2. Type "Code Agent"
3. Or go to Code tab
4. Request: "Create a React login form with validation"
5. Get production-ready code
```
