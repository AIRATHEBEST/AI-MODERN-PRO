## Web Search:

All models automatically:
- Search for latest information
- Pull real-time data
- Access current events
- Find recent updates

Just ask naturally:
- "What happened today in..."
- "Latest news about..."
- "Current price of..."
- "Recent updates on..."

## Creating ZIP Files:

Models can't create actual ZIP files, but they can:
- Generate code to create ZIPs
- Provide instructions for zipping
- List files to include
- Write scripts for compression

Example: "Write a Python script to create a ZIP file with these files..."
